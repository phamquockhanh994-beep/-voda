# ВОДА · fon_audio_player

> Play YouTube audio with the screen off — iOS & Android, no app install needed.

A single-file HTML web app that keeps YouTube audio running in the background using Picture-in-Picture mode. Add videos to a queue, switch tracks from the lock screen, and turn off your display — sound keeps playing.

**Works on:** Safari (iOS 14+) · Chrome (Android 8+)  
**Install:** Open in Safari → Share → Add to Home Screen
